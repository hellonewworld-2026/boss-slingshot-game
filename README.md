# 絶望の上下水道局 ～ワンマン局長の引っ張り決裁アクション～

ちびっこギャング（水道局長）となり、定時退社を企む社員たちに始末書ハンコを押し付ける、スリングショット型のオフィスディフェンス・アクションゲームです。

## 特徴

*   スマホ片手操作（親指一本の引っ張りアクション）に完全対応。
    
*   ピュアな HTML5 Canvas + JavaScript による軽量・高速60FPS動作。
    
*   物理エンジン（弾性反射）をゼロから実装。
    
*   iOSの自動再生制限を突破する Audio アンロック機構搭載。
    

## 遊び方

1.  スマホのブラウザでアクセスします。
    
2.  「決裁を開始する」をタップするとゲームスタート（音が出ます）。
    
3.  局長（ボス）を指でタッチして下方向に引っ張り、離すと突撃します。
    
4.  エレベーター（画面上部）に逃げようとする社員に体当たりして、強制残業させましょう。
    
5.  壁やデスクの反射を利用して、効率よく社員を一網打尽にしてください！
    
6.  画面を素早くダブルタップすると、手榴弾ボムが発動します。
